Class Name: Event_Manager

Responsibilities:
* Tells Calendar the event the user wants to add
* Tells Calendar to make custom event and add it
* Makes recommended event and passes to Event_UI
* Deletes event
* Adds course event and make event object with course information also, it will call Make_Delete_Event to add to database

Collaborators:
* Custom_Event
* Make_Delete_Event
* Event
* Event_UI
