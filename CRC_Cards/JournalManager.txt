Class Name: JournalManager

Responsibilities:
* Stores an instance of FileGateway as accessFiles and an instance of Journal as journal
* Calls accessFiles to create a text file for the journal entry with given title, entry, date and tags and
 stores it in user's journal directory.Also calls journal to store title of entry to file created.
* Calls accessFiles to deletes journal entry file with given title within user's journal directory. Also calls journal to
delete that mapping of these entries' title to its entry file
* Calls on journal to get the journal entry file with the given title. Calls on acess files
to return the information within the journal entry such as title, tags and contents
* Calls journal to get the title of all entries o and returns it as an ArrayList<String>



Collaborators:
* Journal
* FileGateway
